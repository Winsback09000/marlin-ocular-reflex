# Ocular reflex analysis - Marlins

Statistical analysis of ocular muscle tonus during roll rotation in juvenile marlins.

## Data
Experimental dataset (1983), 100 individuals from two Pacific populations.

## Analysis
- Exploration of the periodic signal as a function of roll angle
- Mixed-effects models to account for repeated measurements within individuals

## Reproducibility
The complete analysis pipeline is provided. Reproducibility is ensured for users with access to the original dataset.
